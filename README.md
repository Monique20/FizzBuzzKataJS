Fizz Buzz Kata

The Kata

Return “Fizz”, “Buzz” or “FizzBuzz”.

For a given natural number greater zero return

•	“Fizz” if the number is divisible by 3

•	“Buzz” if the number is divisible by 5

•	“FizzBuzz” if the number is divisible by both 3 and 5
